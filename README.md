## About

This little example shows how to build a firefox extension using Yew and wasm-pack. 

## 🚴 Usage

### 🛠️ Build

When building for the first time, ensure to install dependencies first.

```
yarn install
```

```
yarn run build
```

### 🔬 Serve locally

```
yarn run start:dev
```

### 🦊 Run as a plugin

```
yarn run web-ext
```
